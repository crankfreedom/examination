# ExamHub 用户端 PRD

> **文档版本：** v1.2.0
> **最后更新：** 2026-07-13

---

## 一、概述

本文档定义 ExamHub V1 用户端的产品需求，包括 13 个页面的功能规格、用户角色、操作流程与边界条件。

### 1.1 产品范围

V1 用户端定位为**资料库 + PDF 销售平台**，用户可浏览、搜索、购买和下载考试资料。

### 1.2 核心路径

\\\
游客 → 首页浏览 → 商品详情页（预览） → 支付页（微信扫码） → 自动注册/登录 → 下载页 → 下载 PDF
\\\

## 三、13 个页面详细需求

### 3.1 首页

**核心目标**：展示平台核心价值，引导用户浏览和搜索

| 区域 | 内容 | 数据来源 | 备注 |
|------|------|---------|------|
| 顶部导航 | Logo、搜索框、登录/注册、用户头像 | - | 搜索框吸顶，滚动后固定 |
| 搜索栏 | 搜索输入框 + 搜索按钮 | - | 占位文字："搜索试卷、合集…" |
| 考试分类导航 | 国考 / 省考 / 事业单位 / 教师 / 考研 / 四六级 | 后台配置 | 图标 + 文字，点击跳转分类页 |
| Banner | 推荐活动/限时优惠轮播 | 后台配置 | 可点击跳转商品详情 |
| 最新上传 | 最近上传的试卷/合集卡片列表 | 按上传时间倒序 | 显示封面、标题、价格、下载量 |
| 热门下载 | 下载量最高的资料卡片列表 | 按下载量排序 | 可配置 7 天/30 天/全部 |
| 免费专区 | 价格为 0 元的资料列表 | 商品价格 = 0 | 入口明显，鼓励用户注册 |
| 官方资讯 | 最新考试资讯（考试时间/报名入口/职位表） | 后台发布 | 显示标题和发布日期，最多 5 条 |
| 专题合集 | 运营推荐的专题合集 | 后台配置 | 如"2024 国考真题合集" |
| VIP 专区 | VIP 专属资料列表 | VIP 商品 | 非 VIP 用户显示锁图标 |
| 底部 | 关于我们、友情链接、备案信息 | - | - |

**状态 / 边界**：
- 无内容时显示"暂无数据"占位
- Banner 无配置时隐藏轮播区域
- 加载中显示 skeleton 骨架屏

### 3.2 分类页

**核心目标**：多维筛选，帮助用户快速找到目标资料

**筛选维度**：

| 筛选维度 | 选项 | UI 控件 |
|---------|------|---------|
| 考试类型 | 国考 / 省考 / 事业单位 / 教师 / 考研 / 四六级 | 标签组 |
| 年份 | 2024 / 2023 / 2022 / … | 下拉/标签 |
| 地区 | 北京 / 上海 / 广东 / … | 下拉/标签 |
| 科目 | 行测 / 申论 / 职测 / 综应 / 英语 / 政治 / … | 标签组 |
| 资料类型 | 历年真题 / 模拟卷 / 高频考点 / 专项训练 / 行测 / 申论 | 标签组 |
| 价格区间 | 免费 / 1-10 元 / 10-50 元 / 50 元以上 | 标签组 |
| 难度 | 基础 / 中等 / 困难 | 标签组 |
| 题量 | 1-50 题 / 50-100 题 / 100 题以上 | 标签组 |

**页面布局**：
- 左侧/顶部：筛选区域
- 右侧/下方：卡片网格列表
- 排序：最新 / 最热 / 价格从低到高 / 价格从高到低
- 支持关键词搜索（在当前分类范围内）

**状态 / 边界**：
- 筛选结果为空时显示"没有找到符合条件的资料，建议减少筛选条件"
- 已选筛选条件以标签形式展示，支持一键清除
- URL 同步筛选参数，支持分享和浏览器后退

### 3.3 搜索页

**核心目标**：快速检索并定位资料

**功能规格**：
- 搜索范围：试卷标题、合集标题、商品标题、试题内容、知识点
- 搜索结果分为两组：
  1. 试卷 / 合集 / 商品（核心结果）
  2. 资讯 / 公告 / 知识点（辅助结果）
- 搜索框支持自动补全（热门搜索词推荐）
- 搜索结果支持与分类页相同的多维筛选
- 无结果时显示搜索建议

**状态 / 边界**：
- 搜索词为空时显示搜索历史 + 热门搜索
- 搜索词过短（少于 2 字符）不触发搜索
- 支持拼音/模糊匹配（V2）

### 3.4 商品详情页

**核心目标**：展示资料信息，促使用户购买

**页面结构**：

| 区域 | 内容 |
|------|------|
| 左侧 | 试卷封面/标题、考试类型/年份/地区/科目标签、资料简介、目录树、在线阅读入口 |
| 右侧 | 价格信息（原价/会员价/积分价）、购买按钮、VIP 提示 |
| 下方 | 题目试看区域（前 3~5 题，含答案和解析）、相关推荐 |

**状态 / 边界**：
- 已购买用户：显示"立即下载"按钮，不显示价格
- VIP 用户：显示"VIP 免费"标签
- 商品已下架：显示"该商品已下架"，隐藏购买按钮
- 限时活动：显示倒计时 + 活动价格
- 免费商品：显示"免费下载"按钮
- 加载失败：显示重试按钮

### 3.5 在线阅读页（Reader）

**核心目标**：提供流畅的在线阅读体验

**三栏布局**：

| 区域 | 内容 |
|------|------|
| 左侧（窄） | 题号目录，高亮当前题目，点击跳转 |
| 中间（宽） | 题目内容（题干、选项/填空/主观题） |
|  | 【材料题】材料原文展示：material_group 类型下，题目组上方先展示材料内容（阅读文本/图表/听力文本），再依次列出各题 |
| 右侧（窄） | 答案 + 解析（游客/注册用户仅前 3~5 题可见） |

**功能规格**：
- 支持键盘导航（← → 切换题目）
- 支持字号调整
- 支持切换分享/专注模式
- 支持收藏题目
- 支持切换深色模式
- 支持大图查看（图片题）
- 支持公式渲染（MathJax / KaTeX）

**状态 / 边界**：
- 游客/注册用户看到第 6 题时，右侧解析区域显示"购买后可查看完整解析"
- 长题干支持折叠/展开
- 图片加载失败显示替代文本

### 3.6 支付页

**核心目标**：简化支付流程，降低弃单率

**流程**：
\\\
选择商品 → 确认订单信息（商品名/价格/数量） → 选择支付方式 → 扫码支付
  → 支付成功 → 自动注册账号（游客） → 自动登录 → 跳转下载页
\\\

**支付方式（V1）**：
- 微信扫码支付（主要）

**关键设计**：
- 游客支付即注册：微信返回 openid → 自动创建账号 → 绑定订单 → 生成 JWT → 自动登录
- 用户几乎无感知，后台已完成注册流程
- 支付成功页面显示"立即下载"按钮 + 下载指引

**状态 / 边界**：
- 支付超时：订单保留 30 分钟，超时取消
- 支付失败：显示错误原因和建议操作
- 支付中：轮询订单状态，显示 loading
- 重复支付：自动退款处理

### 3.7 个人中心

**核心目标**：用户信息管理和入口导航

**包含**：
- 用户头像（默认生成）、昵称、手机号、绑定微信
- 等级标识（普通用户 / 已购买用户 / VIP）
- VIP 到期时间（VIP 用户）
- 入口导航：订单中心、下载中心、收藏中心、浏览历史、VIP 中心、积分、设置
- 退出登录

### 3.8 VIP 中心

**核心目标**：展示 VIP 权益，引导购买

**内容**：
- VIP 权益列表（无限下载 / 全部解析 / VIP 专区 / 在线刷题 / AI 功能）
- 价格展示（199 元/年）
- 立即开通按钮
- VIP 到期倒计时（已开通用户）

### 3.9 订单中心

**核心目标**：管理用户的所有订单

**列表展示**：
- 订单号、商品名、金额、支付状态、下单时间
- 状态筛选：全部 / 待支付 / 已支付 / 已退款
- 待支付订单显示"去支付"按钮
- 已支付订单显示"去下载"按钮
- 订单详情（点击展开或跳转）

**状态 / 边界**：
- 无订单时显示"暂无订单"
- 待支付订单超过 30 分钟自动取消

### 3.10 下载中心

**核心目标**：管理已购资料的下载

**功能规格**：

| 区域 | 内容 |
|------|------|
| 顶部 | 筛选标签：全部 / 下载中 / 已下载 / 已更新 |
| 列表 | 资料封面、标题、下载次数/总次数、版本号、更新时间 |
| 操作 | 下载按钮、更新按钮（有新版本时） |

**下载限制逻辑**：
- 普通已购用户：30 天内最多 10 次下载
- VIP 用户：无限下载
- 下载次数耗尽：显示"下载次数已用完，升级 VIP 或下月重置"
- 下载计数规则：每次点击下载按钮计 1 次，重复下载同一文件重复计数

**状态 / 边界**：
- 下载中：显示进度条
- 下载失败：显示重试按钮
- 有新版本：显示"更新"标签 + 更新按钮
- 未购买资料不显示在列表中

### 3.11 收藏中心

**核心目标**：管理用户收藏的资料

**功能规格**：
- 收藏列表（封面、标题、价格、收藏时间）
- 取消收藏
- 已下架商品显示"已下架"标签
- 收藏夹分类（可选 V2）

**状态 / 边界**：
- 无收藏时显示"还没有收藏的资料"
- 最大收藏数量限制（如 200 个）

### 3.12 官方资讯页

**核心目标**：发布与考试相关的官方通知和信息

**内容类型**：
- 考试时间公告
- 报名入口通知
- 职位表发布
- 考试大纲更新
- 成绩查询通知
- 官方通知

**功能规格**：
- 文章列表（标题、发布日期、摘要、浏览量）
- 文章详情（富文本渲染）
- 按分类筛选
- 搜索

### 3.13 官方导航页

**核心目标**：集中列出各考试官方网址

**内容**：
- 按考试类型分类（国考/省考/事业单位/教师/考研/四六级）
- 每个分类下列出：
  - 官方网站名称
  - 官网 URL
  - 功能说明（报名入口 / 成绩查询 / 职位表等）
- 全部由后台维护，前端只做展示

## 四、全局功能

### 4.1 暗黑模式

- V1 即支持
- 跟随系统设置和手动切换
- 所有页面适配暗色主题

### 4.2 响应式适配

- 手机（< 768px）：单栏布局，底部 Tab 导航
- 平板（768px - 1024px）：两栏布局
- PC（> 1024px）：三栏/完整布局

### 4.3 SEO

- 每个页面有独立的 Title / Description / Keywords
- 试卷、合集、商品页面有独立永久 URL
- 生成 Sitemap 和 Robots.txt
