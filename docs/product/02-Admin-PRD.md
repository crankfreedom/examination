# ExamHub 管理端 PRD

> **文档版本：** v1.2.0
> **最后更新：** 2026-07-13

---

## 一、概述

本文档定义 ExamHub V1 管理端的产品需求，包括 13 个管理模块的功能规格、操作流程与权限体系。

### 1.1 产品范围

管理端用于平台运维人员管理内容、订单、用户、系统配置等全部后台事务。

### 1.2 管理端入口

- URL：\/admin\ 路径
- 独立登录页（账号密码登录，不支持微信扫码）
- 需要管理员角色权限

---

## 二、管理角色与权限

| 角色 | 权限范围 |
|------|---------|
| 客服 | 订单管理、用户管理（只读 + 客服消息） |
| 管理员 | 内容管理、商品管理、订单管理、用户管理、资讯管理、SEO 管理（不含系统设置和权限管理） |
| 超级管理员 | 全部权限，包括系统设置、权限中心 |

---

## 三、13 个管理模块详细需求

### 3.1 仪表盘

**核心目标**：今日运营数据概览

| 指标 | 说明 | 刷新频率 |
|------|------|---------|
| 今日订单数 | 今日成功支付订单数 | 实时 |
| 今日收入 | 今日总支付金额（元） | 实时 |
| VIP 总人数 | 当前有效 VIP 用户数 | 每日 |
| 新增资料数 | 今日新增试卷数 | 实时 |
| 今日下载次数 | 今日 PDF 下载总次数 | 实时 |
| 最近 7 天订单趋势 | 折线图 | 每日 |
| 热门资料 Top 10 | 下载量最高的 10 个资料 | 每日 |
| 系统状态 | 服务器状态、存储用量 | 实时 |

### 3.2 题库管理

**核心目标**：管理所有题目数据

**可管理字段**：

| 字段 | 类型 | 说明 |
|------|------|------|
| 题型 | 枚举 | 单选题 / 多选题 / 判断题 / 填空题 / 主观题 |
| 题干 | 富文本 | 支持文字、图片、公式 |
| 选项 | JSON | A/B/C/D/E/F… 支持图片选项 |
| 正确答案 | JSON | 支持单选、多选、判断对错、填空答案 |
| 解析 | 富文本 | 支持文字、图片、公式 |
| 所属材料 | 关联 | 可选关联 Material |
| 知识点 | 标签 | 多标签 |
| 难度 | 枚举 | 基础 / 中等 / 困难 |
| 来源 | 枚举 | 历年真题 / 模拟题 / 原创题 |
| 年份 | 数字 | 题目所属年份 |
| 地区 | 字符串 | 题目所属地区 |
| 考试类型 | 枚举 | 国考 / 省考 / 事业单位 / 教师 / 考研 / 四六级 |
| 状态 | 枚举 | 草稿 / 已发布 / 已废弃 |

**操作**：
- 新增题目（表单 / JSON 编辑两种模式）
- 编辑题目
- 批量导入（通过 Importer）
- 批量修改（批量设置考试类型/年份/地区等）
- 导出为 JSON
- 预览题目渲染效果
- 搜索/筛选（支持多条件组合）

**状态 / 边界**：
- 题目删除为逻辑删除（标记已废弃）
- 已关联试卷的题目，修改后版本号 +1
- 图片/公式上传

### 3.3 试卷管理

**核心目标**：将题目组装为试卷

**功能规格**：
- 新建试卷 → 填写基本信息（标题、考试类型、年份、地区、科目、难度、说明）
- 添加题目 → 从题库搜索/筛选 → 勾选 → 添加（支持题目编号批量添加）
- 拖拽排序 → 调整题目顺序
- 支持分部分（如：第一部分 言语理解、第二部分 数量关系…）
- 自动统计：总题数、各题型数量、各部分题数
- 预览试卷渲染效果
- 试卷状态：草稿 / 已发布 / 已废弃

**状态 / 边界**：
- 试卷至少包含 1 题
- 试卷发布后锁定编辑（需创建新版本）
- 试卷修改后版本号 +1

### 3.4 合集管理

**核心目标**：将多张试卷组合为合集

**功能规格**：
- 新建合集 → 填写基本信息（标题、考试类型、说明、封面图）
- 添加试卷 → 从试卷库搜索/筛选 → 勾选 → 添加
- 拖拽排序
- 自动统计：试卷总数、总题数
- 合集状态：草稿 / 已发布 / 已废弃

### 3.5 商品管理

**核心目标**：管理商品的定价和销售策略

**功能规格**：

| 字段 | 类型 | 说明 |
|------|------|------|
| 商品名称 | 字符串 | 展示给用户的名称 |
| 商品描述 | 富文本 | 详细商品介绍 |
| 关联资源 | 数组 | 可绑定多个 Paper / Collection |
| 封面图 | 图片 | 商品封面 |
| 定价 | 数字 | 原价（元） |
| VIP 价 | 数字 | VIP 专享价 |
| 积分价 | 数字 | 积分兑换所需积分 |
| 商品类型 | 枚举 | 普通商品 / VIP 专属 / 积分兑换 |
| 限时活动 | 是否限时 | 开始时间、结束时间、活动价 |
| 状态 | 枚举 | 草稿 → 待发布 → 已发布 → 已下架 → 已删除 |

**操作**：
- 新建 / 编辑 / 上下架 / 删除
- 复制商品（快速创建相似商品）
- 批量上下架
- 限时活动直接在后台设置（不改代码）

**状态 / 边界**：
- 商品价格可设为 0（免费资源）
- 已下架商品：前端隐藏，已购用户仍可下载
- 已删除商品：彻底不可见
- 一个商品可绑定多个 Paper / Collection

### 3.6 PDF 管理

**核心目标**：管理 PDF 的生成和版本

**功能规格**：
- 查看已生成的 PDF 列表（关联的 Paper / Collection、生成时间、文件大小、版本号）
- 手动触发 PDF 生成（按需生成）
- 查看生成日志（成功/失败/耗时）
- PDF 版本管理（每次内容更新生成新版本）
- 手动清除 PDF 缓存（触发重新生成）

**状态 / 边界**：
- PDF 生成异步执行，后台队列处理
- 生成失败有重试机制

### 3.7 订单中心

**核心目标**：管理所有订单

**列表展示**：

| 字段 | 说明 |
|------|------|
| 订单号 | O000001 |
| 用户 | 用户昵称/手机号 |
| 商品 | 购买的商品名称 |
| 金额 | 实付金额 |
| 支付方式 | 微信支付 |
| 状态 | 待支付 / 已支付 / 已退款 / 已取消 |
| 下单时间 | - |
| 支付时间 | - |

**操作**：
- 查看订单详情
- 退款处理（原路退回）
- 手动补单（特殊情况）
- 订单导出为 Excel/CSV
- 查看订单的下载记录

**状态 / 边界**：
- 已支付订单不支持修改金额
- 退款需二次确认

### 3.8 用户中心

**核心目标**：管理所有用户

**列表展示**：

| 字段 | 说明 |
|------|------|
| 用户编号 | U000001 |
| 昵称 | - |
| 手机号 | - |
| 微信绑定 | 是否绑定 |
| 注册时间 | - |
| 最后登录 | - |
| VIP 到期 | 到期日期（如有） |
| 积分 | 当前积分 |
| 状态 | 正常 / 已封禁 |

**操作**：
- 查看用户详情（订单 / 下载 / 收藏 / 登录记录）
- 手动开通 / 续费 / 取消 VIP
- 手动调整积分
- 封禁 / 解封用户
- 发送系统消息
- 搜索用户

### 3.9 内容中心

**核心目标**：管理资讯、公告、友情链接等前端展示内容

**管理内容**：

| 模块 | 说明 |
|------|------|
| 资讯管理 | 发布/编辑/删除官方资讯（标题、摘要、正文、分类、封面） |
| 公告管理 | 发布/编辑/删除系统公告 |
| 官方网址 | 添加/编辑/删除各考试官方网址（名称、URL、说明、考试分类） |
| 友情链接 | 添加/编辑/删除友情链接（名称、URL、Logo） |
| Banner 管理 | 添加/编辑/删除首页 Banner（图片、链接、排序、启用/禁用） |
| 首页推荐 | 配置首页推荐资料（最新上传、热门下载、专题合集等） |

### 3.10 SEO 管理

**核心目标**：优化搜索引擎展示

**管理内容**：

| 模块 | 说明 |
|------|------|
| 页面 SEO | 为每个页面设置 Title / Description / Keywords |
| URL 管理 | 固定 URL 不可修改，仅展示 |
| Sitemap | 自动生成，支持手动触发刷新 |
| Robots.txt | 配置搜索引擎爬取规则 |
| 404 页面 | 配置自定义 404 |
| 友链优化 | — |

**核心目标**：配置系统全局参数

**配置项**：

| 配置组 | 配置项 |
|--------|--------|

### 3.12 权限中心（RBAC）

**核心目标**：精细化管理管理员权限

**功能规格**：

| 模块 | 说明 |
|------|------|
| 管理员列表 | 查看、新增、编辑、删除管理员账号 |
| 角色管理 | 新建/编辑角色，配置角色名称、描述 |
| 权限配置 | 按菜单 + 按钮粒度配置权限（增/删/改/查/导出） |
| 角色分配 | 为管理员分配角色 |
| 操作日志 | 记录管理员所有操作（谁、什么时间、做了什么） |

**权限粒度**：
- 菜单级：显示/隐藏某个管理模块
- 操作级：增/删/改/查/导出/导入 每个模块
- 按钮级：显示/隐藏特定按钮

### 3.13 数据采集（Chalk）

**核心目标**：自动化采集外部题库网站试题数据

**功能规格**：

| 功能 | 说明 |
|------|------|
| 触发采集 | 管理员设置采集套数，调用 `/chalk/create/examination` 接口启动自动化爬取 |
| 采集来源 | 粉笔网等第三方题库网站（通过 Selenium WebDriver 自动化操作） |
| 采集内容 | 题干、选项、正确答案、解析、考点、来源、答题统计、材料题内容 |
| 图片处理 | 自动截图题目中的图片，以 UUID 命名保存到本地 |
| 数据保存 | 采集结果保存为本地 JS 文件（CommonJS 模块格式），作为中间产物 |
| 后续导入 | 采集结果通过 Importer 转换为统一 JSON Schema 后导入内容中心 |

**操作**：
- 设置采集套数（num）并触发采集任务
- 查看采集任务状态（启动/进行中/完成）
- 查看采集历史和结果文件
- 将采集结果导入内容中心（通过 Importer 流程）

**状态 / 边界**：
- 采集任务在独立子进程中运行，不影响主服务
- 第三方网站凭证直接书写在 config/chalk.ts 中（参考 config/env.ts）
- 采集的数据需经管理员审核后方可发布
- 采集操作记录到操作日志

## 四、全局要求

### 4.1 UI 风格
- 简洁专业的管理后台风格
- 左侧导航菜单，顶栏面包屑 + 用户信息
- 暗黑模式支持
- 响应式（至少适配平板和 PC）

### 4.2 数据导出
- 订单列表、用户列表支持导出 CSV / Excel
- 可自定义导出字段

### 4.3 操作日志
- 所有关键操作记录日志
- 可查询、导出

### 4.4 通知
- 新订单通知
- 系统异常通知
- 低频发送，避免打扰
