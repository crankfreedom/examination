## 一、用户端核心流程

### 1.1 访客购买下载流程

\\\
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌───────────┐
│  首页/   │ ──→ │ 商品详情  │ ──→ │  支付页   │ ──→ │  下载中心  │
│  搜索页   │     │  /预览   │     │  (扫码)   │     │  (PDF下载) │
└──────────┘     └──────────┘     └──────────┘     └───────────┘
                                          │
                                          ▼
                                    ┌──────────┐
                                    │ 微信扫码  │
                                    │  支付    │
                                    └────┬─────┘
                                         │
                                    ┌────▼─────┐
                                    │ 自动注册  │
                                    │ + 自动登录 │
                                    │ + 绑定订单 │
                                    └──────────┘
\\\

**涉及页面**：首页 → 商品详情页 → 支付页 → 下载中心

**分支情况**：
- 已登录用户：跳过自动注册和自动登录
- 已购买用户：直接跳转下载中心
- VIP 用户：显示 VIP 价格，可直接下载

### 1.2 在线阅读流程

\\\
┌──────────┐     ┌──────────┐     ┌──────────────┐     ┌──────────┐
│ 商品详情  │ ──→ │ 在线阅读  │ ──→ │ 题目导航 +   │ ──→ │ 查看答案  │
│  /试看   │     │  /阅读   │     │ 题目内容     │     │  /解析   │
└──────────┘     └──────────┘     └──────────────┘     └──────────┘
                                          │
                                          ▼
                                    ┌──────────────┐
                                    │ 游客/注册用户  │
                                    │ 仅前 3~5 题   │
                                    │ 第 6 题起锁定 │
                                    │  → 引导购买   │
                                    └──────────────┘
\\\

**涉及页面**：商品详情页 → 在线阅读页

### 1.3 搜索筛选流程

\\\
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│ 首页搜索  │ ──→ │ 搜索页   │ ──→ │ 筛选维度  │ ──→ │ 结果列表  │
│  /导航    │     │ (关键词) │     │ (多选)    │     │ (排序)   │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
                                      │
                                      ▼
                                ┌──────────┐
                                │ 切换到    │
                                │ 分类页    │
                                └──────────┘
\\\

**涉及页面**：首页 → 搜索页 / 分类页

### 1.4 VIP 开通流程

\\\
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│ VIP 中心  │ ──→ │ 选择套餐  │ ──→ │  支付    │ ──→ │ VIP 特权 │
│  入口     │     │ (年费)   │     │  (扫码)  │     │  解锁    │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
\\\

**涉及页面**：VIP 中心 → 支付页

## 二、管理端核心流程

### 2.1 内容录入流程

\\\
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│ 新建题目  │ ──→ │ 录入题干  │ ──→ │ 设置答案  │ ──→ │ 预览确认  │
│  (表单)   │     │ /选项    │     │ /解析    │     │ (渲染)   │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
                                                      │
                                                      ▼
                                                ┌──────────┐
                                                │ 发布/保存  │
                                                │  草稿     │
                                                └──────────┘
\\\

### 2.2 批量导入流程（Importer）

\\\
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│ 选择文件  │ ──→ │ 上传文件  │ ──→ │ 解析预览  │ ──→ │ 检查修改  │ ──→ │ 确认导入  │
│ JSON/MD/ │     │          │     │ (发现    │     │ (逐题    │     │ (批量    │
│ DOCX     │     │          │     │  错误)   │     │  修改)   │     │  写入DB) │
└──────────┘     └──────────┘     └──────────┘     └──────────┘     └──────────┘
                                                                          │
                                                                          ▼
                                                                    ┌──────────┐
                                                                    │ 生成报告  │
                                                                    │ 成功/失败 │
                                                                    │ 题数统计  │
                                                                    └──────────┘
\\\

### 2.3 试卷构建流程

\\\
┌──────────────┐     ┌──────────────────┐     ┌──────────┐     ┌──────────┐
│ 新建试卷      │ ──→ │ 定义部分结构       │ ──→ │ 拖拽排序  │ ──→ │ 发布试卷  │
│  (元信息)     │     │ regular /         │     │ 分部分    │     │          │
└──────────────┘     │ material_group     │     └──────────┘     └──────────┘
                     └────────┬─────────┘
                              │
              ┌───────────────┼──────────────────┐
              ▼               ▼                  ▼
     ┌────────────────┐  ┌──────────────┐  ┌──────────────┐
     │ regular 部分    │  │ material_    │  │ 搜索/筛选     │
     │ 从题库选题      │  │ group 部分   │  │ 题目编号      │
     └────────────────┘  │ 先选材料      │  └──────────────┘
                          │ (material)   │
                          │ 再选关联题目   │
                          └──────────────┘

**材料题分组说明**：
- 部分类型支持 regular（常规部分）和 material_group（材料题组）两种 type
- 选择 material_group 类型时，需先选择材料（material），再在该材料下选择关联题目
- paper_questions.material_group_index 标记同一材料组内的题目，用于前端渲染时展示材料原文
\\\

### 2.4 商品发布流程

┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐

│  (名称/   │     │ Paper/   │     │ /VIP价   │     │ (上架)   │
│  描述/图) │     │ Collect  │     │ /限时    │     │          │
└──────────┘     └──────────┘     └──────────┘     └──────────┘

│           │     │          │     │  确认)   │     │  退回)   │

## 六、异常 / 边界页面

| 场景 | 用户端 | 管理端 |
|------|--------|--------|
| 404 | 显示"页面不存在"，返回首页 | 显示"页面不存在"，返回仪表盘 |
| 403 | — | "权限不足"提示 |
| 500 | 显示"服务器繁忙，请稍后重试" | 显示错误信息 + 操作 ID |
| 网络离线 | 离线提示 + 重试按钮 | 离线提示 |
| 空数据 | "暂无数据" + 操作引导 | "暂无数据" |
| 加载中 | Skeleton 骨架屏 | Loading 指示器 |
