# ExamHub V1 路线图 — 资料库 + PDF 销售平台

> **文档版本：** v1.2.0
> **最后更新：** 2026-07-13

---

## 一、V1 定位

**资料库 + PDF 销售平台** — 用户可浏览、搜索、购买和下载考试资料（PDF）。

---

## 二、V1 核心目标

| 维度 | 目标 |
|------|------|
| 试卷数量 | 上线时 >= 100 套 |
| 覆盖考种 | >= 6 种（国考/省考/事业单位/教师/考研/四六级） |
| 核心功能 | 浏览 / 搜索 / 购买 / 支付(微信) / 下载 / 水印保护 |
| 管理端 | 13 个管理模块完整可用 |
| 稳定性 | 可用性 >= 99.5%，PDF 生成成功率 >= 99.5% |

## 五、V1 不做的功能

| 功能 | 原因 | 计划版本 |
|------|------|---------|
| 在线刷题 | V1 只做阅读模式 | V2 |
| AI 解析 | 需要 NLP 能力积累 | V3 |
| AI 组卷 | 需要算法积累 | V3 |
| AI 错题分析 | 需要用户数据积累 | V3 |
| 手机 APP | Web + PWA 足够覆盖 | 后续评估 |
| 多语言 | V1 只做中文 | 后续评估 |
| 社交功能 | 分散核心目标 | 后续评估 |
