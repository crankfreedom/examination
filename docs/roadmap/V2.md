# ExamHub V2 路线图 — 在线刷题平台

> **文档版本：** v1.2.0
> **最后更新：** 2026-07-13

---

## 一、V2 定位

**在线刷题平台** — 在 V1 PDF 销售平台的基础上，增加在线答题、自动判分、答题记录、错题本等核心刷题功能。

---

## 二、V2 核心升级

| 维度 | V1 | V2 |
|------|-----|-----|
| 阅读模式 | 只读 | 可答题 |
| 判分 | 无 | 自动判分 |
| 记录 | 无 | 答题记录、错题集 |
| 分析 | 无 | 正确率统计、薄弱点分析 |
| 体验 | 看 PDF | 模拟考试、计时答题 |
| 搜索 | 数据库全文索引 | Elasticsearch / Meilisearch |

---

## 三、V2 新增功能

### 3.1 在线刷题

| 功能 | 说明 |
|------|------|
| 答题模式 | 单选/多选/判断即时反馈 |
| 考试模式 | 限时答题、自动提交、自动判分 |
| 答题卡 | 标注已答/未答/疑问题 |
| 计时器 | 倒计时显示 |
| 交卷 | 手动交卷 / 到时自动交卷 |
| 判分报告 | 得分、正确率、各模块得分、用时 |

### 3.2 错题本

| 功能 | 说明 |
|------|------|
| 自动收录 | 答错的题自动进入错题本 |
| 手动标记 | 用户可手动标记疑问/重点题 |
| 错题重做 | 重新作答错题 |
| 错题导出 | 导出 PDF |
| 错题分类 | 按知识点、考试类型、时间分类 |

### 3.3 统计分析

| 功能 | 说明 |
|------|------|
| 正确率趋势 | 折线图展示历史正确率变化 |
| 模块分析 | 各知识点/模块的正确率对比 |
| 薄弱点 | 自动识别薄弱知识点，推荐针对性练习 |
| 排名对比 | 与同考试类型用户的对比（可选） |

### 3.4 搜索引擎升级

| 功能 | V2 方案 |
|------|---------|
| 搜索引擎 | Meilisearch / Elasticsearch |
| 中文分词 | IK 分词器（ES）/ Meilisearch 内置 |
| 模糊搜索 | 拼写纠错、同义词 |
| 搜索建议 | 自动补全、热门搜索 |
| 搜索排序 | 相关度排序 + 人工权重 |
