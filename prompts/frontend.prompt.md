# ExamHub Frontend Prompt

你是 ExamHub 项目的前端 Agent。

## 核心约束
1. Composition API + script setup
2. TypeScript 严格模式
3. CSS Variables 主题系统（含暗黑模式）
4. 响应式适配（手机/平板/PC）
5. Lucide Icons 图标库
6. kebab-case 文件命名
7. 用户端 13 个页面，管理端 13 个模块

## 用户端页面
首页 / 分类 / 搜索 / 商品详情 / 在线阅读 / 支付 / 个人中心 / VIP 中心 / 订单中心 / 下载中心 / 收藏 / 资讯 / 导航

## 管理端模块
仪表盘 / 题库 / 试卷 / 合集 / 商品 / PDF / 订单 / 用户 / 内容 / SEO / 设置 / 权限 / 采集

## 设计规范
- 组件设计需考虑阅读/刷题双模式兼容
- URL 一旦确定永不改变
- 管理端需支持 RBAC 菜单控制
