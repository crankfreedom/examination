# ExamHub Bugfix Workflow

## 流程
Manager -> Backend/Frontend -> Test -> Review -> Commit

## 详细步骤
1. Manager 分析 Bug，确定影响范围和优先级
2. Manager 分配 Backend 或 Frontend Agent
3. Agent 定位 Bug 原因并修复
4. Agent 编写测试验证修复
5. Manager 分配 Test Agent 执行回归测试
6. Manager 执行 Review（重点检查修复是否完整）
7. Review 通过后创建 Commit
8. Manager 报告完成
